# 项目简介

## 核心功能
- 基于 YOLOv8 的 3D 打印缺陷检测系统
- 提供 692 张标注数据集，包含 1 类缺陷（fail）
- 支持图片、视频及摄像头输入的缺陷检测
- 界面实时显示目标位置、目标总数和置信度信息
- 支持批量检测并查看所有结果
- 允许自定义置信度和 IOU 阈值
- 可选择显示标签和原图
- 支持多检测模型选择
- 提供检测结果保存功能

## 适用场景
- 3D 打印质量检测与质量控制
- 工业制造缺陷自动识别
- 产品质量监控流程
- 研究与教育领域应用

## 技术特点
- 基于 Python 3.9 开发
- 集成 OpenCV、PyQt5、Ultralytics 和 Torch 1.9 等技术栈
- 提供完整的源代码、UI 界面源文件和数据集
- 包含训练好的模型及运行教程
- 支持多平台环境配置（PyCharm/VSCode + Anaconda）
